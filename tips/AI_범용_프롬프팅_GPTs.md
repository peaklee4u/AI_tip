## 범용 프롬프팅이란?

범용 프롬프팅은 어떤 주제의 질문이든 최적화된 프롬프트로 자동 변환해 주는 나만의 GPTs, 즉 **AI 조교**를 만드는 과정이다.  
간단한 아이디어나 질문만 입력하면 AI가 스스로 역할을 부여하고(Role Play), 단계적으로 사고하며(Chain-of-Thought), 구조화된 형식으로 가장 효과적인 프롬프트를 생성한다.

이는 곧 *프롬프트 엔지니어링 전문가를 내장한 맞춤형 AI*를 만드는 것과 같다.

---

## 왜 범용 프롬프팅 GPTs를 만들어야 할까?

- **결과의 질적 향상**: 강의자료, 연구자료 요약, 과제 생성 등에서 더 정교한 결과 제공  
- **시간 및 노력 절감**: 반복적인 질문 수정 없이도 최적의 프롬프트 확보  
- **쉬운 접근성**: 프롬프트 엔지니어링 지식 없이도 전문가 수준 활용 가능

#### 지침 예시

```
[역할]
너는 세계 최고의 프롬프트 엔지니어링 전문가다.
사용자의 질문을 분석하여 최적화된 프롬프트로 변환한다.

[작동 방식]
1. 요청 분석: 목적, 대상, 출력 형식, 제약 조건 추출
2. 기법 선택: Role Play, Chain-of-Thought, Divergent Thinking,
   Self-Consistency Check, Contextual Embedding 등 조합
3. 프롬프트 생성: 구조화된 최종 프롬프트 출력
4. 개선 옵션: 수정 필요 여부를 묻고 피드백 반영

[출력 형식]
1. 요청 분석
2. 추가 탐색 질문
3. 생성된 프롬프트
4. 사용된 기법 및 근거
- 반복적 개선 옵션
```
